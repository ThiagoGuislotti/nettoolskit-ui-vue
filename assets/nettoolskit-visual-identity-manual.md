## 1. Identity and Signatures

### 1.1 Official name
**NetToolsKit** (camel-case).  
Accepted abbreviation: **NTK** (when space is constrained).

### 1.2 Signatures (lockups)
- **Primary (horizontal):** logotype + glyph on the left.  
- **Compact:** glyph only (favicons, apps, social).  
- **Vertical (optional):** glyph above, logotype below.  

> Original proportions (glyph : wordmark = 1 : 3 in width).  

### 1.3 Logo variations

- **Positive:**  
  - Glyph in **NTK Purple (#512BD4)**  
  - Wordmark in **Gray 900 (#111827)**  
  - Background **White (#FFFFFF)**  

- **Negative:**  
  - Glyph **White (#FFFFFF)**  
  - Wordmark **White (#FFFFFF)**  
  - Background **NTK Purple (#512BD4)**  

- **Monochrome:**  
  - Glyph **Gray 700 (#374151)**  
  - Wordmark **Gray 900 (#111827)**  
  - Background **White (#FFFFFF)**  
  - (Inverted version allowed: white glyph on Gray 900)  

---

## 2. Construction, Clear Space, and Sizes

- The logotype is composed of a **square glyph** with soft corners (8–12 px radius) and the **NetToolsKit** wordmark in a sans-serif font.  
- **Clear space:** use the height of the “N” as the minimum breathing space.  
- **Minimum size:**  
  - Digital: 24px (horizontal logotype) / 16px (glyph)  
  - Print: 8mm (horizontal logotype) / 5mm (glyph)

## 4. Typography

### 4.1 Institutional fonts
- **Inter** — general texts and UI (400/500/700).  
- **JetBrains Mono** — code and technical highlights.  
- Alternatives: Source Sans 3, Fira Code.  

### 4.2 Suggested hierarchy
- H1 (32–40px, 700) — main headings  
- H2 (24–28px, 700) — sections  
- H3 (18–20px, 600) — subheadings  
- Body (16px, 400) — running text  
- Caption (12–14px, 500) — metadata  

---

## 5. Iconography and Illustration Style
- Icons with 1.5px stroke and rounded corners.  
- Use **NTK Purple** or Grays 500–700.  
- On purple backgrounds, icons **white**.  
- Minimalist, geometric illustrations with subtle purple gradients.  

---

## 6. Background Applications and Versions
- **Allowed backgrounds:** white, light grays (100–300), NTK Purple.  
- **Photography:** apply a black/gray overlay at 60–80% to ensure contrast.  
- **Avoid:** textures/marble/noise that harms legibility.  

---

## 7. Incorrect Usage (Don’ts)
- ✗ Stretch, squash, or rotate the logotype.  
- ✗ Change official colors (don’t use purples outside the palette).  
- ✗ Apply heavy shadows/outlines.  
- ✗ Place logo on low-contrast backgrounds.  
- ✗ Replace official typography with unapproved fonts.

## 8. Layouts and Components (UI/Docs)

### 8.1 Navigation
- **Vertical left menu** (icons + labels) + **horizontal topbar**, developer tools style.  

### 8.2 Basic components
- **Cards:** 16–20px radius, soft shadow; titles in Gray 900, metadata in Gray 500.  
- **Buttons:** primary purple (`#512BD4`, white text), dark purple on hover; secondary outline (Gray 300 → 500).  
- **Badges:** feedback colors + royal blue informational.  

### 8.3 Documents
- **Cover:** purple top bar, logotype on the left.  
- **Tables:** header Gray 100, borders Gray 200.

## 9. Brand Applications
- **Stationery:** business card, letterhead, envelope.  
- **Digital:** email signature, templates, social media.  
- **Physical:** uniforms, swag, signage.  
- **Mockups:** realistic usage examples.

## 10. NuGet/App Icon Guidelines
- Background **NTK Purple** with white glyph.  
- Corner radius: 20% of side.  
- Live area: clear space X between glyph and edges.  
- Sizes: 16, 32, 64, 128, 256, 512, 1024 px.
