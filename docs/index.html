Why NetToolsKit UI?

Built for developers who need consistent, accessible, and themeable components.

Theme System

Runtime theme switching with CSS variables. Includes Sentinela, PlaTEA, and Dark themes out of the box.

Form Components

Complete form toolkit with BaseInput, BaseSelect, BaseMultiSelect, BaseTextarea, and date/time pickers.

Layout Components

Flexible layouts with BaseHeader, BaseSidebar, BaseHero, BaseSection, and BaseFooter components.

Composables

Reusable logic with useTheme, useNotification, useDialog, useFormRules, useDebounce, and more.

Accessibility

WCAG compliant components with keyboard navigation, ARIA labels, and focus management.

Design Tokens

Consistent spacing, shadows, and transitions defined as CSS variables and SCSS mixins.

Simple to Use

Import components and composables with a single entry point. Full TypeScript support included.

  • ✓ Single entry point export for all components
  • ✓ Full TypeScript definitions included
  • ✓ Quasar-first CSS with design tokens
  • ✓ Runtime theme switching 
// Import components
import { 
  BaseInput,
  BaseSelect,
  MetricCard
} from '@nettoolskit/ui-vue'

// Import composables
import { 
  useTheme,
  useNotification
} from '@nettoolskit/ui-vue'

// Use in your component
const { currentTheme, setTheme } = useTheme()
const { notify } = useNotification()

setTheme('sentinela')
notify('Theme changed!', 'success')

Get Started

Add NetToolsKit UI to your project in minutes.

1

Add as Submodule

git submodule add https://github.com/ThiagoGuislotti/nettoolskit-ui-vue.git src/shared/ui
2

Import Styles

@import '@/shared/ui/styles/design-system.scss';
3

Use Components

import { BaseInput } from '@/shared/ui';
